# Biometric-Based Secure Authentication for Cloud Services

## 👤 Author: Preethi Reddy

## 📘 Overview
This project demonstrates a secure authentication mechanism using **biometric fingerprint recognition** to access cloud-based services. It simulates both authentication and resource servers using Java and MySQL.

## 🚀 Features
- Fingerprint-based user login
- Two-server architecture: Authentication Server & Resource Server
- MySQL database integration
- Simulated secure cloud access

## 🛠️ Tech Stack
- Java (J2EE)
- MySQL
- JDBC
- NetBeans / Eclipse
